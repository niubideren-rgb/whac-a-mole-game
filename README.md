# 🎮 打地鼠游戏 (Whac-A-Mole)

基于 React + TypeScript + Vite 开发的经典打地鼠游戏，符合 Quest 任务 #2332 需求规范。

## 📋 项目信息

- **任务 ID**: 2332
- **任务标题**: 打地鼠游戏开发
- **组织**: BTC (V98HXB9)
- **技术栈**: React 18 + TypeScript + Vite

## 🎯 功能特性

### 核心玩法
- ✅ 3×3 = 9 个地鼠洞
- ✅ 地鼠随机弹出，停留时间可配置
- ✅ 点击击中得分（普通地鼠 10 分，金色地鼠 20 分）
- ✅ 5 个关卡，难度递增
- ✅ 每关 60 秒倒计时
- ✅ 达到目标分数进入下一关

### 游戏界面
- ✅ **启动界面**: 游戏说明、关卡信息、开始/退出按钮
- ✅ **游戏界面**: 关卡/分数/时间显示、3×3 地鼠洞、暂停/退出按钮
- ✅ **暂停界面**: 当前状态显示、继续/退出选项
- ✅ **关卡完成**: 得分统计、进入下一关
- ✅ **游戏结束**: 失败原因、统计数据、重新开始
- ✅ **胜利界面**: 通关祝贺、最终统计、准确率

### 音效系统
- ✅ Web Audio API 生成音效（无需外部资源）
- ✅ 击中地鼠音效
- ✅ 金色地鼠特殊音效
- ✅ 失误点击音效
- ✅ 关卡完成音效
- ✅ 游戏结束/胜利音效
- ✅ 音效开关控制

### 响应式设计
- ✅ 手机端优化（< 480px）
- ✅ 平板端优化（481px - 768px）
- ✅ 桌面端自适应
- ✅ 深色模式支持

## 🚀 快速开始

### 环境要求

- Node.js >= 20.0.0
- npm >= 9.0.0

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动本地开发服务器，访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

输出目录：`dist/`

### 预览生产构建

```bash
npm run preview
```

## 🎮 游戏说明

### 操作方式
- **点击地鼠**: 击中地鼠获得分数
- **暂停按钮**: 暂停游戏
- **退出按钮**: 结束游戏
- **音效开关**: 开启/关闭音效

### 得分规则
- 🐹 普通地鼠：10 分
- 🌟 金色地鼠：20 分
- ❌ 点击空地：失误（不计分）

### 胜利条件
- 每关 60 秒内达到目标分数
- 完成全部 5 个关卡即为通关
